# Portal Digital PCM Genteng & Lazismu

Dokumentasi ini menjadi blueprint backend untuk fase fondasi project.

## Dokumen

- `backend-structure-architecture.md`
  Acuan utama struktur `app/`, pembagian domain, naming convention, service layer, dan best practice yang siap dipakai sebagai standar implementasi.

- `backend-foundation-plan.md`
  Blueprint arsitektur backend, struktur folder, modular breakdown, naming convention, dan best practice.

- `backend-domain-database-design.md`
  Desain domain, model utama, relasi data, dan daftar migration yang perlu dibuat.

- `backend-implementation-roadmap.md`
  Urutan implementasi, package yang dibutuhkan, workflow development, dan milestone sampai campaign engine berjalan.

## Prinsip Utama

- Monolith modular
- Service layer
- Clean code
- Config-driven campaign engine
- Production-oriented
- Siap dikembangkan ke multi-cabang / SaaS di masa depan
